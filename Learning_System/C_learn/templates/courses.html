Introduction

This course provides a comprehensive introduction to the fundamentals of computer programming using the C language. Designed for beginners, it focuses on the core concepts of programming, including problem-solving techniques, logic building, and the development of efficient algorithms. The C language, known for its performance and close-to-hardware operations, serves as an excellent foundation for learning structured programming and basic software development.

Topics covered

  • Introduction to Programming and C Language Basics
  • Control Structures
  • Functions
  • Arrays and Strings
  • Pointers and Memory Management
  • Structures and Unions
  • File Handling
  • Introduction to Data Structures

Course Learning Outcomes

By the end of this course, students will be able to:

  • Understand the basic structure and syntax of C programs and apply these to write simple programs.
  • Utilize control structures to make decisions and manage program flow efficiently.
  • Develop modular programs using functions and understand parameter passing.
  • Work with arrays, pointers, and dynamic memory allocation to manage data efficiently.
  • Apply structures to create complex data types and manage data in files.
  • Understand and implement basic data structures and algorithms in C.
  • Develop problem-solving skills by creating efficient and scalable algorithms using the C language.

Technical Requirements

This course is delivered entirely online. You will be required to have access to a computer or web-capable mobile device and have consistent access to the internet to either view or download the necessary course resources and to attempt any auto-graded course assessments and the final exam.
To access the full course, including assessments and the final exam, you will need to be logged into your Learning System account and enrolled in the course. Introduction

This course introduces students to Object-Oriented Programming (OOP) concepts using the C++ programming language. Building on the foundations of procedural programming, this course focuses on creating modular, reusable, and efficient code using the OOP paradigm. C++ serves as an ideal language to explore key OOP principles such as encapsulation, inheritance, and polymorphism, while also leveraging powerful features like operator overloading and templates.

Topics covered

  • Introduction to C++ and OOP Concepts
  • Classes and Objects
  • Constructors and Destructors
  • Inheritance
  • Polymorphism
  • Encapsulation and Abstraction
  • Friend Functions and Operator Overloading
  • Templates and Exception Handling
  • File Handling in C++
  • Introduction to Standard Template Library (STL)

Course Learning Outcomes

By the end of this course, students will be able to:

  • Understand and apply object-oriented programming principles such as encapsulation, inheritance, and polymorphism in C++.
  • Define and manipulate classes and objects, utilizing access control and constructors effectively.
  • Implement and understand various types of inheritance and the role of virtual functions in achieving polymorphism.
  • Overload operators and functions to customize object behavior and write more efficient code.
  • Apply templates and exception handling to write generic and robust code.
  • Perform file handling in C++ using object-oriented principles for persistent data storage.
  • Use the Standard Template Library (STL) to write efficient, reusable, and modular C++ programs.
